## Scripts

### Development

To run locally: 
```
npm run dev
```

To lint code:
```
npm run lint
```

To run tests:
```
npm run test
```

To check test coverage:
```
npm run coverage
```

### Production
To create a production build:
```
npm run build
```

To preview the production build:
```
npm run preview
```
